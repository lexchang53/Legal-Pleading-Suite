# 法院電子卷證去浮水印工具 (pdf-watermark-remover)

本專案是一個專為無損清理法院電子卷證（包含下載的電子卷證與閱卷室掃描卷宗）中「四個角落的閱卷人姓名文字浮水印」、「底部的閱卷系統防偽及動態時間戳記字串」，以及自法院閱卷室掃描而成的「四個角落天平圖片浮水印」而設計的自訂 Skill 與獨立 CLI 工具。本工具亦支援自動解除 PDF 唯讀與編輯限制密碼。

---

## 📖 技術原理與開發歷程

在開發此工具時，我們經歷了以下技術探索與驗證：

1. **邊緣座標物理 Redaction 擦除法（文字浮水印）**：
   - 經分析 PDF 文字塊（text blocks）的座標系統，我們發現所有浮水印字串（角落姓名與底部防偽時間）的 Y 座標都落在紙張的最邊緣（頂部與底部邊緣範圍內，預設值均為 `45.0` pt），而正文內容絕不會印在這麼邊緣的位置。
   - **設計實作**：利用 `PyMuPDF (fitz)` 掃描每一頁，只要發現文字塊的座標在邊緣區間內，就直接套用 PDF 標準的 **Redaction (編輯擦除)**，從內容流中徹底抹除該字元的操作符，且完全不破壞背景或其他正文。

2. **XObject 圖片解綁與指令清理（天平圖片浮水印）**：
   - 自法院閱卷室掃描出的 PDF 在四個角落會有小型天平圖片浮水印。我們透過解析 PDF 內部 XObject 圖片資源，安全地將角落的天平小圖片解綁，並在該頁的內容流 (Content Streams) 中抹除其繪製指令 (`/name Do` 運算子)，實現無損清除背景上的圖片浮水印。

3. **自動解鎖密碼限制**：
   - 法院電子卷證通常會施加編輯限制與唯讀密碼。本工具在讀取 PDF 時，若發現有權限限制，會自動對其進行無損解鎖，並在儲存時寫出已解密、無密碼限制的檔案。

## 🏃 使用說明

### 命令行執行
您可以在終端機中直接呼叫 Python 腳本來處理您的 PDF 檔案：

```bash
python scripts/remove_watermarks.py -i "[輸入的 PDF 檔案路徑]" -o "[輸出的 PDF 檔案路徑]"
```

*可選參數：*
* `-t` 或 `--top`：頂部邊緣判定範圍 (pt)，預設 `45.0`。
* `-b` 或 `--bottom`：底部邊緣判定範圍 (pt)，預設 `45.0`。
* `-f` 或 `--freq`：角落姓名浮水印的頁面重複頻率閾值，預設 `3`（大於此閾值的姓名才會被判定為浮水印，避免誤刪內容）。

### AI 技能呼叫
當本 Skill 安裝在全域 `skills` 目錄後，您只需在與 Antigravity 的對話中提及：
> 「幫我移除這個 PDF 的浮水印：`[檔案路徑]`」

系統便會自動在背景執行此指令並為您處理完畢！
