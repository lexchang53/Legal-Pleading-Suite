# PDF 邊緣浮水印物理去水印技能 (pdf-watermark-remover)

本技能專為無損清理法院電子卷證（包含下載的電子卷證與閱卷室掃描卷宗）中「四個角落的閱卷人姓名浮水印」與「底部的閱卷平台防偽及時間戳記字串」，以及自法院閱卷室掃描而成的 PDF 四個角落「天平圖片浮水印」而設計。本技能亦支援無損解除密碼限制。

## 📖 觸發場景
當使用者提及以下關鍵字或需求時，應觸發此技能：
* "幫我把這個 PDF 的浮水印拿掉"、"清理 PDF 的浮水印"
* "去除電子卷證的浮水印"
* "移除 PDF 角落的姓名"
* "處理電子卷證"、"去除閱卷浮水印"
* "移除天平浮水印"、"清理掃描卷宗浮水印"

## 🛠️ 依賴工具
* Python 3
* `PyMuPDF` 套件 (即 `fitz`)
* 本專案隨附之核心腳本位於技能目錄下的 `scripts/remove_watermarks.py`

---

## 🏃 使用步驟與指令

當此技能被觸發時，AI (你) 應執行以下步驟：

### 1. 確定輸入與輸出路徑
向使用者確認 PDF 檔案路徑。若使用者僅提供輸入檔案，預設在同目錄下產生檔名後綴為 `_無浮水印.pdf` 的輸出檔案。

**🔴 CHECKPOINT / STOP**：在呼叫 Python 腳本前，必須向使用者確認輸入檔案與預期的輸出路徑。若輸入檔案不存在，應引導使用者提供正確的絕對路徑。

### 2. 呼叫背景 Python 腳本
*   **第一步：依賴庫自動檢查與安裝**
    在執行去水印命令前，應確保 `PyMuPDF` 庫已正確安裝。若執行腳本出錯並提示 `ModuleNotFoundError: No module named 'fitz'`，AI 應自動先執行以下指令以安裝依賴：
    ```bash
    pip install pymupdf
    ```
*   **第二步：呼叫處理腳本**
    找出本技能安裝的實際目錄路徑，利用 `run_command` 執行以下指令：

    ```bash
    python "[pdf-watermark-remover 技能目錄的絕對路徑]/scripts/remove_watermarks.py" --input "[輸入檔案絕對路徑]" --output "[輸出檔案絕對路徑]"
    ```

    *若有特殊邊緣範圍判定需求，可透過 `--top` 與 `--bottom` 調整範圍（預設值均為 `45.0` pt）。*
    *若檔案被密碼鎖定（唯讀或有限制），此腳本會自動以空密碼將其無損解鎖，並儲存為已解密且無密碼限制的版本。*

*   **第三步：錯誤恢復與 Fallback 處理**
    *   若執行腳本回傳失敗，且輸出含有密碼錯誤，請告知使用者該檔案受到強密碼保護，需請使用者先提供解密密碼，或手動解密後再行處理。
    *   若因其他 Python 環境問題導致腳本崩潰，請提供詳細的錯誤日誌並引導使用者手動疑難排解。

### 3. 回報處理進度與結果
執行完成後，告知使用者清理了多少處文字浮水印（包含角落時間、姓名）與多少個圖片浮水印（包含天平），並提供清理後 PDF 的絕對路徑供使用者點擊開啟確認。

---

## 🚫 限制與反例
*   **不要**在未備份原始檔案的情況下，直接覆寫輸入檔案。
*   **不要**對邊緣含有重要簽名、蓋印、正式公文字號或非浮水印資訊的文件直接執行，避免誤抹除重要資訊。
*   本工具專為法院電子卷證之邊緣浮水印（頂底 45 pt 內）與角落天平小圖片設計。**不要**將此工具套用於正文中央有大面積浮水印、滿版背景文字或非電子卷證類型之檔案，此舉效果有限且極易誤抹除正常文字。
