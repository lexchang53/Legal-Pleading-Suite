# PDF 邊緣浮水印物理去水印技能 (pdf-watermark-remover)

本技能專為無損清理法院電子卷證（包含下載的電子卷證與閱卷室掃描卷宗）中「四個角落的閱卷人姓名浮水印」與「底部的閱卷平台防偽及時間戳記字串」，以及自法院閱卷室掃描而成的 PDF 四個角落「天平圖片浮水印」而設計。本技能亦支援無損解除密碼限制。

## 📖 觸發場景
當使用者提及以下關鍵字或需求時，應觸發此技能：
* "幫我把這個 PDF 的浮水印拿掉"、"清理 PDF 的浮水印"
* "去除電子卷證的浮水印"
* "移除 PDF 角落的姓名"
* "處理電子卷證"、"去除閱卷浮水印"
* "移除天平浮水印"、"清理掃描卷宗浮水印"

## 🛠️ 依賴工具
* Python 3
* `PyMuPDF` 套件 (即 `fitz`)
* 本專案隨附之核心腳本位於技能目錄下的 `scripts/remove_watermarks.py`

---

## 🏃 使用步驟與指令

當此技能被觸發時，AI (你) 應執行以下步驟：

### 1. 確定輸入與輸出路徑
向使用者確認 PDF 檔案路徑。若使用者僅提供輸入檔案，預設在同目錄下產生檔名後綴為 `_無浮水印.pdf` 的輸出檔案。

### 2. 呼叫背景 Python 腳本
找出本技能安裝的實際目錄路徑，利用 `run_command` 執行以下指令：

```bash
python "[pdf-watermark-remover 技能目錄的絕對路徑]/scripts/remove_watermarks.py" --input "[輸入檔案絕對路徑]" --output "[輸出檔案絕對路徑]"
```

*若有特殊邊緣範圍判定需求，可透過 `--top` 與 `--bottom` 調整範圍（預設值均為 `45.0` pt）。*
*若檔案被密碼鎖定（唯讀），此腳本會自動將其無損解鎖並儲存為無密碼版本。*

### 3. 回報處理進度與結果
執行完成後，告知使用者清理了多少處文字浮水印（包含角落時間、姓名）與多少個圖片浮水印（包含天平），並提供清理後 PDF 的絕對路徑供使用者點擊開啟確認。
